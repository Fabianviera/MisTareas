# MisTareas

**MisTareas** es una aplicación de escritorio para gestionar listas de tareas, desarrollada en Python con customtkinter.

📖 **[Documentación completa](https://fabianviera.github.io/MisTareas)**

---

## Características

- Añadir, completar y eliminar tareas
- Registro de fecha y hora de creación y completado
- Opción "siempre visible" (pin) para mantener la ventana encima de otras
- Menú contextual rápido
- Diseño limpio en tonos azules y grises
- Datos guardados automáticamente en local

## Instalación

### Windows
[⬇ Descargar Installer_MisTareas.exe](https://github.com/Fabianviera/MisTareas/releases/download/v0.2-beta/Installer_MisTareas.exe)

Ejecuta el instalador y sigue el asistente. Se creará acceso directo en el escritorio y en el menú inicio.

> **⚠ Aviso de Windows SmartScreen**
> Al ejecutar el instalador, Windows puede mostrar el mensaje "Windows protegió su equipo".
> Esto ocurre porque el archivo no está firmado digitalmente. Para continuar:
> 1. Haz clic en **"Más información"**
> 2. Haz clic en **"Ejecutar de todas formas"**

## Uso

1. Escribe una tarea en el campo de texto y pulsa **+** o **Enter**
2. Marca la casilla para completarla
3. Pulsa **✕** para eliminarla
4. Usa el botón **📌** para fijar la ventana encima de las demás
5. Usa el botón **⋮** para acceder al menú rápido

MisTareas

MisTareas es una aplicación de escritorio para gestionar listas de tareas, desarrollada en Python con customtkinter.

Características

  • Añadir, completar y eliminar tareas
  • Registro de fecha y hora de creación y completado
  • Opción "siempre visible" (pin) para mantener la ventana encima de otras
  • Menú contextual rápido
  • Diseño limpio en tonos azules y grises
  • Datos guardados automáticamente en local

Instalación

Windows

Descarga el instalador Installer_MisTareas.exe y sigue el asistente. Se creará acceso directo en el escritorio y en el menú inicio.

macOS

En construcción...

Abre el archivo .dmg, arrastra la app a la carpeta Aplicaciones.

Uso

  1. Escribe una tarea en el campo de texto y pulsa + o Enter
  2. Marca la casilla para completarla
  3. Pulsa ✕ para eliminarla
  4. Usa el botón 📌 para fijar la ventana encima de las demás
  5. Usa el botón ⋮ para acceder al menú rápido

Tecnologías

  • Python 3.10
  • customtkinter
  • PyInstaller (compilación)
  • Inno Setup (instalador Windows)
